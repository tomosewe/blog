Setting up a blog with Hugo and Netlify

Thu, Jan 25, 2018

Hugo is a static site generator that claims to be the world’s fastest framework for building websites. I’ve used jekyll in the past to set up a quick blog but found it clunky to use and the build time to be quite slow. So far Hugo has been really easy to set up. (apart from a few issues getting Hugo installed on Ubuntu correctly) I’m loving the huge amount of themes to choose from and switch between.

Deploying a new Hugo site is also really easy. In the past I’ve used GitHub Pages and also Firebase for hosting. However this time I’ve decided to use Netlify as I’ve heard quite a lot of great things about it. There are heaps of great tutorials I’ve found for deploying a Hugo site to Netlify.

This blog post is not yet complete. I’ve got to try and deploy it to Netlify first before I can finish it off.
